# Roadmap

## Phase 1 — Kenya MVP

- Static Astro site
- Leaflet map
- OpenStreetMap tiles
- Kenya GeoJSON data
- Search and filters
- Location cards and popups
- Contribution pages and issue template
- Data validation script

## Phase 2 — Public launch

- Verify all Kenya seed data
- Add official source links to every entry
- Add screenshot and demo GIF to README
- Add GitHub Actions validation
- Publish on GitHub Pages, Netlify or Vercel

## Phase 3 — Multi-country expansion

- Add Uganda, Rwanda, Tanzania and Nigeria
- Add country switcher
- Add per-country counts
- Add regional maintainers

## Phase 4 — Ecosystem intelligence

- Add public reports generated from the data
- Add city pages
- Add category pages
- Add downloadable CSV export

## Phase 5 — API and admin

- Add API endpoint or static JSON export
- Add CMS/admin if pull requests become a bottleneck
- Add moderation workflow
